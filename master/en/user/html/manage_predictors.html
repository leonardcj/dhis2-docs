10.10. Manage predictors

10.10.1. About predictors

Predictors are data elements automatically generated from samples of current and past data. You use predictors to analyse previous data in order to predict future values, for example for disease surveillance.

A predictor specifies an expression and a set of *sample constraints* (identical to the current implementation of validation rules). This includes a period type, organisation unit levels, and parameters specifying past or periodic sample windows.

In the Maintenance app, you manage the following predictor objects:

Table 10.17. Predictor objects in the Maintenance app

Object type

Available functions

Predictor

Create, edit, clone, delete, show details and translate


10.10.2. Create a predictor

  1. Open the Maintenance app and click Other > Predictor.

  2. Click the add button.

  3. In the Name field, type the predictor name.

  4. (Optional) In the Code field, assign a code.

  5. (Optional) Type a description.

  6. Select a Period type.

  7. Assign organisation unit levels.

    Tip

    If you're missing organisation unit level, you can create new ones.

  8. Create a Generator:

  9. Click Sample skip test and create an expression which is used to exclude data from the generation process.

    The expression should evaluate to true or false.

  10. Enter a Sequential sample count.

  11. Enter an Annual sample count.

  12. Enter a Sequential skip count.

  13. Click Save.

10.10.3. Edit metadata objects

  1. Open the Maintenance app and find the type of metadata object you want to edit.

  2. In the object list, click the options menu and select Edit.

  3. Modify the options you want.

  4. Click Save.

10.10.4. Clone metadata objects

  1. Open the Maintenance app and find the type of metadata object you want to clone.

  2. In the object list, click the options menu and select Clone.

  3. Modify the options you want.

  4. Click Save.

10.10.5. Delete metadata objects

  1. Open the Maintenance app and find the type of metadata object you want to delete.

  2. In the object list, click the options menu and select Delete.

  3. Click Confirm.

10.10.6. Display details of metadata objects

  1. Open the Maintenance app and find the type of metadata object you want to view.

  2. In the object list, click the options menu and select Show details.

10.10.7. Translate metadata objects

  1. Open the Maintenance app and find the type of metadata object you want to translate.

  2. In the object list, click the options menu and select Translate.

  3. Select a locale.

  4. Type a Name, Short name and Description.

  5. Click Save.

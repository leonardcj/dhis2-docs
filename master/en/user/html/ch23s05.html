23.5. Edit events in grid

Events are displayed in a tabular way. Each row corresponds to an event. The columns display information about the events, more specifically they display data for data elements which are part of the program.

The Edit in grid function allows you to edit the selected event within the table but only those columns (data elements) visible in the grid. If you need more columns, use Show/hide columns to speficy which columns should be displayed in the list.

  1. In the Apps menu, click Event Capture.

  2. Select an organisation unit.

  3. Select a program.

    All events registered to the selected program show up in a list.

  4. Click the event you want to modify and select Edit in grid.

  5. Modify the event details.

  6. Click on another event to close the edit mode.
